# Quiz App with React & TypeScript

[See the page](https://quiz-typescript.vercel.app/)

* I made a Quiz App with React & TypeScript.
* I used Styled Components for styling.

### I used;
  - <b>TypeScript</b>
  - <b>Styled Components</b>
  - <b>Context API</b>
